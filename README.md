# 🎮 XO Game (Tic-Tac-Toe)

A simple and interactive Tic-Tac-Toe (XO) game built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features

- 🎲 Two-player mode (X vs O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Restart game button
- 📱 Responsive design
- 🎨 Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
